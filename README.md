LIMINALE
Una stanza digitale non è uno spazio.
È una condizione.
Qui non esistono istruzioni.
Solo presenze.
Ogni passaggio lascia una direzione invisibile.
Le direzioni si accumulano.
Lo spazio ascolta.
Finché le visioni restano compatibili, tutto sembra immobile.
Quando smettono di esserlo, la stanza cambia forma.
Non c’è un momento preciso in cui qualcosa accade.
L’evento emerge.
Non modifico lo spazio.
Rendo visibile il peso delle scelte.
